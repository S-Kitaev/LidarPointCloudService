# Технические характеристики

В данном документе указаны основные технические характеристики системы съёмки и визуализации 3D облаков точек (LiDAR)

---

|---------|---------|
| Ширина установки | 340 мм  |
| Длина установки | 370 мм  |

| Полная высота (установка + тренога) | 1260 мм |

| Параметр | Значение |
|---------|----------|

---

## 3. Характеристики LiDAR (RPLIDAR A1M8)

| Параметр | Значение |
|---------|----------|
| Питание | 3.3 В |
| Частота вращения | 8 Гц |
| Максимальная дальность измерения | 10 м |
| Минимальная дальность измерения | 50–70 мм |
| Скорость сбора точек | 1200 ± 200 точек/с |
| Доля точек с немаксимальным качеством | ≤ 3 % *(в типовых условиях)* |

---

> **Ограничения сканирования**:  
> LiDAR **не регистрирует** или **даёт искажённые данные** при:  
> 1. Темных (поглощающих ИК) поверхностях;  
> 2. Прозрачных материалах (стекло, плёнка);  
> 3. Зеркальных отражающих поверхностях.
