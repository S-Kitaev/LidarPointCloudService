## API Endpoints

### Public
| Method | Endpoint          | Description          |
|--------|------------------|----------------------|
| GET    | `/`              | Home (Anon)          |
| GET    | `/registration`  | Registration Form    |
| POST   | `/registration`  | Registration Web     |
| GET    | `/login`         | Login Form           |
| POST   | `/login`         | Login Web            |

---

### User Pages
| Method | Endpoint                   | Description    |
|--------|----------------------------|----------------|
| GET    | `/{user_id}`               | Home User      |
| GET    | `/{user_id}/create`        | Create Cloud   |
| GET    | `/{user_id}/check`         | Check Cloud    |
| GET    | `/{user_id}/check/{experiment_id}` | View Cloud |
| GET    | `/{user_id}/connect`       | Connect Cxd    |
| GET    | `/{user_id}/capture`       | Capture        |
| GET    | `/{user_id}/docs/specs`    | Docs github hrefs     |
| GET    | `/{user_id}/docs/setup`    | Docs inner hrefs      |
| GET    | `/{user_id}/docs/setup/{doc_name}`    | Doc md file   |

---

### User API
| Method | Endpoint                                                                  | Description         |
|--------|---------------------------------------------------------------------------|---------------------|
| GET    | `/{user_id}/api/experiments`                                              | Get Experiments     |
| GET    | `/{user_id}/api/experiments/{experiment_id}/measurements`                | Get Measurements    |
| POST   | `/{user_id}/create/save`                                                  | Insert Data         |
